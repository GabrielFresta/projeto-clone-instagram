# 📸 Clone do Instagram

Uma recriação simplificada da interface do Instagram desenvolvida para praticar conceitos fundamentais de desenvolvimento web front-end.

🚀 **[Clique aqui para ver o projeto rodando na prática](https://gabrielfresta.github.io/projeto-clone-instagram/)**

---

## 🛠️ Tecnologias Utilizadas

- *HTML5*: Estruturação semântica da página.
- *CSS3*: Estilização, layout responsivo e alinhamentos (Flexbox/Grid).
- *JavaScript (Vanilla)*: Manipulação do DOM e lógica de interatividade.
- *localStorage*: Armazenamento local de dados no navegador.

---

## 📌 Funcionalidades

- Interface responsiva inspirada no feed e elementos do Instagram.
- Persistência de dados simples utilizando localStorage.
